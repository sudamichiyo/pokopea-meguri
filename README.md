# pokopea-meguri
# 実行方法
```
$ git clone git@github.com:sudamichiyo/pokopea-meguri.git

// mainブランチにいる場合
$ git switch feature/video-display

$ go run main.go

ブラウザでhttp://127.0.0.1:5500/frontend/index.htmlにアクセス

検索欄に沖縄を入力して検索(現状YouTubeの動画が再生されるのは沖縄のアメリカンビレッジのリンクのみ)

アメリカンビレッジの方のリンクをクリックすると別タブでYouTubeを開き動画が再生される
